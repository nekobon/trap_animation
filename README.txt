Quadrupole ion trap animation with an ion

Creates images to make an animation of an trapped ion. 
This program is not intended to scientifically simulate
an ion trap, but to demonstrate the basic idea of the trap.

Files:
 - trap_potential_animation.py
   Main program.
 - caotic.gif
   An example output with an ion not trapped well
 - stable.gif
   An example output with an ion trapped well

Requirements:
 Requires python OpenGL library.
